You just need to join all previous hints, return the resulted string and wrap your code into function expression. Using type hints for input and output data are highly recommended.

Here is the whole solution:

def longest_palindromic(a: str) -> str:

    l = len(a)
    res = ""
    for s in range(l):
        if l - s <= len(res):
            break
        for e  in range(s + 1, l + 1):
            subs = a[s:e]
            if subs == subs[::-1] and len(subs) > len(res):
                res = subs

    return res

Here is another approach: still double loop but one is for the length of the substring (from bigger, to make it efficient) and the other - for starting index. Then find substring and check if it's palindromic. If so, return it immediately. 

def longest_palindromic(text):

    s = len(text)
    for size in range(s)[::-1]:
        for index in range(s - size):
            word = text[index:index + size + 1]
            if word == word[::-1]:
                return word

Check all possible strings using recursion, by calling the same function for two strings without one border character, and take the longest palindromic. 

def longest_palindromic(text):

    if text == text[::-1]:
        return text

    return max(longest_palindromic(text[:-1]), longest_palindromic(text[1:]), key = len)
